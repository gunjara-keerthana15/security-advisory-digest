# AI Usage Note

This project utilizes Artificial Intelligence techniques through a Retrieval-Augmented Generation (RAG) architecture to process and summarize cybersecurity advisories.

## AI is used for

- Extracting relevant information from uploaded security advisory documents.
- Retrieving contextually relevant content using vector-based semantic search.
- Generating concise and actionable summaries using a Large Language Model (LLM).
- Identifying critical vulnerabilities, affected systems, severity levels, and recommended mitigations.

## Disclaimer

The AI-generated summaries are intended to assist users in quickly understanding security advisories and prioritizing response actions. Final security decisions should be validated by cybersecurity professionals, as AI-generated outputs may occasionally contain inaccuracies or omissions.

## Technologies Used

- Python
- LangChain
- FAISS / Vector Database
- Retrieval-Augmented Generation (RAG)
- Large Language Models (LLMs)
